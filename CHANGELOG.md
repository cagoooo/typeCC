# Changelog - 中文注音打字遊戲 Pro

## [1.2.0] - 2026-03-18
### Added
- **初學者模式 (Beginner Mode)**：為三年級小學生設計，極慢速且掉落稀疏，顯示完整注音與按鍵提示。
- **動態生成控制**：改用定時生成器取代遞迴補位，並根據難度設定畫面字元上限。

### Changed
- **難度平衡優化**：調降「入門」模式基礎速度 20%，減少開局壓力。
- **生成頻率調整**：優化各難度的生成間隔，避免字元瞬間堆疊。

---

## [1.1.0] - 2026-03-18
### Added
- **Firebase 整合**：支援匿名登入與全球排行榜 (Firestore)。
- **成績截圖分享**：整合 `html2canvas` 實作一鍵分享功能。
- **音效系統**：加入打擊音、遊戲結束音與倒數計時警告音。

---

## [1.0.0] - 2026-03-17
### Added
- **專案專業化重構**：從單一 HTML 遷移至 Vite 專案架構。
- **PWA 支援**：實作 Service Worker 離線快取與安裝資訊清單。
- **難度系統**：加入入門、進階、大師三級難度選擇。
- **詞庫擴充**：大幅增加常用字詞與四字成語。
- **GitHub Actions**：設定 CI/CD 自動部署流程。
