# Changelog - 中文注音打字遊戲 Pro

## [1.3.0] - 2026-03-21
### Added
- **虛擬小鍵盤 (Bopomofo Virtual Keyboard)**：新增注音虛擬鍵盤，支援點擊輸入，解決無實體鍵盤的玩家無法遊玩問題。
- **全螢幕模式 (Fullscreen Mode)**：加入全螢幕切換功能按鈕，提供更專注的遊戲體驗。
- **動態太空貓吉祥物 (Dynamic Mascot)**：遊戲中常駐太空貓吉祥物，會隨著打字正確、漏字與待機等狀態變化表情。
- **掉落物難度優化**：增加更多注音提示層級，調整中文詞彙掉落的節奏。

## [1.2.1] - 2026-03-18
### Fixed
- **Firebase 穩定性優化**：降級 Firebase SDK 至穩定版 v10.8.0 並微調初始化順序，徹底解決 Firestore 可能出現的 `Service not available` 報錯。
- **PWA 圖示修復**：更新 `manifest.json` 使用 Google 官方 Noto Emoji 圖示資源，修復連結失效導致的下載錯誤。
- **UI 勘誤清理**：移除全專案中殘留的「代碼內容」文字，並修復遊戲結束畫面的按鈕佈局。

## [1.2.0] - 2026-03-18
### Added
- **初學者模式 (Beginner Mode)**：為三年級小學生設計，極慢速且掉落稀疏，顯示完整注音與按鍵提示。
- **動態生成控制**：改用定時生成器取代遞迴補位，並根據難度設定畫面字元上限。

### Changed
- **難度平衡優化**：調降「入門」模式基礎速度 20%，減少開局壓力。
- **生成頻率調整**：優化各難度的生成間隔，避免字元瞬間堆疊。

---

## [1.1.0] - 2026-03-18
### Added
- **Firebase 整合**：支援匿名登入與全球排行榜 (Firestore)。
- **成績截圖分享**：整合 `html2canvas` 實作一鍵分享功能。
- **音效系統**：加入打擊音、遊戲結束音與倒數計時警告音。

---

## [1.0.0] - 2026-03-17
### Added
- **專案專業化重構**：從單一 HTML 遷移至 Vite 專案架構。
- **PWA 支援**：實作 Service Worker 離線快取與安裝資訊清單。
- **難度系統**：加入入門、進階、大師三級難度選擇。
- **詞庫擴充**：大幅增加常用字詞與四字成語。
- **GitHub Actions**：設定 CI/CD 自動部署流程。
