# 中文注音打字遊戲：使用指南

這是一份關於「中文注音打字遊戲」的詳細使用手冊。本遊戲旨在提升使用者的繁體中文注音輸入速度與準確度。

## 🎮 遊戲玩法

1.  **開始遊戲**：點擊「開始遊戲」按鈕進入遊戲畫面。
2.  **目標**：字元（包含漢字與注音提示）會從畫面上方掉落。您必須在字元觸底前將其「擊落」。
3.  **擊落方式**：
    *   **直接輸入注音碼**：例如看到「的」，輸入 `ㄉㄜ˙`。
    *   **輸入對應按鍵**：使用標準大千式注音鍵盤對應碼（例如「的」對應 `2k7`）。
    *   **空白鍵確認**：對於一聲的字（無聲調符號），最後須按下 **[空白鍵]** 進行確認擊發。
4.  **得分與時限**：
    *   每個成功的字元都會根據掉落速度給予分數加成。
    *   預設遊戲時間為 **180 秒**。

## ⌨️ 注音按鍵對應表

本遊戲採用標準注音符號對應鍵位：

| 類別 | 注音符號 | 對應按鍵 |
| :--- | :--- | :--- |
| **聲母** | ㄅ~ㄌ | 1~x |
| **介母** | ㄧ, ㄨ, ㄩ | u, j, m |
| **韻母** | ㄚ~ㄦ | 8~- |
| **聲調** | ˊ, ˇ, ˋ, ˙ | 6, 3, 4, 7 |
| **一聲** | (無) | **[空白鍵]** |

## ✨ 技術特色

*   **RWD 響應式設計**：支援桌面與行動裝置螢幕。
*   **低延遲體驗**：採用純前端 JavaScript 實作，確保輸入反應即時。
*   **視覺反饋**：擊中字元時會有縮放消失動畫與得分提示。
*   **自動適應**：當視窗縮放時，正在掉落的字元會自動調整水平位置。
